## About

A Kanban-style project management tool similar to Trello. Built to understand drag-and-drop UI, GraphQL APIs, and team collaboration features.

## Tech Stack

React, Go, GraphQL, PostgreSQL

## Getting Started

```bash
git clone https://github.com/shashhii/Project-Management-Tool.git
cd Project-Management-Tool
```
